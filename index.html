Projects

Mixed Feelings Game

A 3D dodging game completed within 3 days as a contender in the UBC Game Jam 2020 competition using C# and Unity. The story of the game depicts the protagontists prevail over their emotional struggles and obstacles.

TA VS Students Game

Single player 2D maze and multiple levels based on a twist on student and teaching assistant interactions as part of and academic project. Built from the ground up using the foundations of Java, Maven, and OOD.

AI Vaccum

A representative way of displaying AI search algorithms processes with a 2D grid and a vaccum trying to clean each dirty room in the enviornment. Showcases BFS, UCS, A*, and DFS in Python. Displays each search algorithms observable states of the enviornment.

Hotel Booking System

A Hotel booking system that takes advantage of Object Oriented Design. Implements a simple server connection to showcase handling of multiple user connections and bookings without causing overlap. Reproduces confirmation and customer information back to user upon succession/fail

Valorant Database

Database project utilizing React, HTML, CSS, Javascript, and Node.js to allow for seamless data entry, retrieval, and manipulation for the game Valorant. The foundation for further insight into data the Game provides and records for further possible analysis such as statistical data.
